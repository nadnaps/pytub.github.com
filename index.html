    Why Python?

    You should learn/use Python for scientific computing because Python

    • is a powerful programming language
    • features millions of modules for everything from FEM to computer algebra
    • code can be as performant as C code
    • is free
    • is blessed by a large community
    • is one of the most popular programming languages

    That's why we

    • set up a lecture series in the upcoming winter term 2014, including Python tools
    • urge staff to install the newest releases at the desktop computers
