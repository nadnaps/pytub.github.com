    Why Python?

    You should learn/use Python for scientific computing because Python

    • is a powerful programming language
    • features millions of modules for everything from FEM to computer algebra
    • code can be as performant as C code
    • is free
    • is blessed by a large community
    • is one of the most popular programming languages

    That's why we

    • have a weekly user meeting on Mondays at MA 464 at 5 PM
    • set up a lecture series in the upcoming summer term 2013
    • urge staff to install the newest releases at the desktop computers

    Start right now by

    • checking the material on the tools seminar homepage
    • dropping by at our user meeting
    • check out Google's online Python Class
    • browsing the extensive python documentation on the web
    • reading a report on the benefits of Python in scientific computing at the university
    • asking questions on stackoverflow
    • joining our mailing list
